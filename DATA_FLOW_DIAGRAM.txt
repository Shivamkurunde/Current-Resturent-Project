[User Database]
  - Stores: User accounts, credentials, profile info
  - Table: users
  - Access: Read/Write

[OTP Store]
  - Stores: OTP codes, expiry times
  - Table: otps
  - Access: Read/Write/Delete

[Session Store]
  - Stores: Active user sessions
  - Type: Server-side session
  - Access: Read/Write/Delete

[Cart Database]
  - Stores: Shopping cart items
  - Table: cart_items
  - Access: Read/Write/Delete

[Order Database]
  - Stores: Order information
  - Table: orders
  - Access: Read/Write/Update

[Order Items Database]
  - Stores: Items in each order
  - Table: order_items
  - Access: Read/Write
